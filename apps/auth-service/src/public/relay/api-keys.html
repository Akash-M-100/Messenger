Security Best Practices

Never expose secret keys in public GitHub repositories or front-end applications. We recommend using environment variables or a dedicated secret management service. Rotate keys every 90 days for maximum security.

Quick Start

curl -X POST https://api.relay.io/v1/broadcast \
  -H "Authorization: Bearer RE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "+1234567890",
    "message": "Hello World"
  }'
const Relay = require('@relay/sdk');
const relay = new Relay('RE_API_KEY');

await relay.broadcast({
  to: '+1234567890',
  message: 'Hello World'
});
